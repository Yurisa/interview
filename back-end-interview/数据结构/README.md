### 数组
#### 数组基础
- 把数据排成一排进行存放
- 数组最大的优点：快速查询。scores[2]
- 数组最好应用于"索引有语意"的情况

- 但也并非所有语意的索引都适用于数组 例如：身份证号：11011122344442
- 数组也可以处理"索引没有语意"的情况
